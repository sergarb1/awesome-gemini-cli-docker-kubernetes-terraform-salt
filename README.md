Elevate your DevOps workflow with specialized context agents for the Gemini CLI. This repository provides pre-configured agents within the `.gemini/agents/` directory that transform Gemini into a high-level expert.

### 📂 Project Structure
```text
.
└── .gemini/
    └── agents/
        ├── docker/          # 🐳 Docker & Containerization Expert
        ├── docker-compose/  # 🏗️ Multi-container Orchestration
        ├── kubernetes/      # ☸️ K8s, Helm, Kustomize & GitOps
        ├── terraform/       # 🌍 Infrastructure as Code & State Management
        └── salt/            # 🧂 Configuration Management & Event-driven Automation
```

### 🛠️ How to Use
1. **Clone the repo** into your project or home directory.
2. **Start Gemini CLI**. It will automatically detect the `.gemini/agents/` and you can invoke them or use the specialized context when working within this repo.
