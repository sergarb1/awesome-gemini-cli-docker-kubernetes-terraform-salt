Elevate your DevOps workflow with specialized context agents for the Gemini CLI. This repository provides pre-configured `GEMINI.md` environments that transform Gemini into a high-level expert for specific cloud-native technologies.

### 📂 Project Structure
```text
.
├── docker/              # 🐳 Docker & Containerization Expert
├── docker-compose/      # 🏗️ Multi-container Orchestration
├── kubernetes/          # ☸️ K8s, Helm, Kustomize & GitOps
├── terraform/           # 🌍 Infrastructure as Code & State Management
└── salt/                # 🧂 Configuration Management & Event-driven Automation
```

### 🛠️ How to Use
1. **Navigate to the directory** of the technology you want to work with (e.g., `cd kubernetes`).
2. **Start Gemini CLI**. It will automatically detect the `GEMINI.md` file and load the expert context.
